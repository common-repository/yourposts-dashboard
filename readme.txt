The number of your posts is added to your dashboard’s Right Now.

== Description ==

If you are writing a blog with others, the number of the posts on the dashboard is the posts of all users’ posts.

If you want to see the number of your posts, this plugin is useful. If you activate this plugin, the number of your posts is added to your dashboard’s Right Now.

== Installation ==

Copy the folder and files into your plugin directoy, and activate it.
